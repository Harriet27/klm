# PT Kreasi Layanan Medis

## Frontend
In the `frontend` directory, you can run:

### `npm install`
Installs all the dependencies.

### `npm start`
Runs the app.

## API
In the `frontend` directory, you can run:

### `npm install`
Installs all the dependencies.

### `nodemon server`
Runs the app.

---

**Note: Node JS and Nodemon should be installed globally through the system**
